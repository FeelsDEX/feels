# Quickstart

## Prerequisites

Before you begin, make sure you have:
- A Solana wallet (Phantom, Solflare, etc.)
- Some SOL for transaction fees
- Basic understanding of DeFi concepts

## Connect Your Wallet

1. Navigate to [feels.exchange](https://feels.exchange)
2. Click "Connect Wallet" in the top right
3. Select your wallet provider
4. Approve the connection request

## Your First Swap

1. Select the token you want to swap from
2. Enter the amount
3. Select the token you want to receive
4. Review the swap details including:
   - Exchange rate
   - Price impact
   - Minimum received
5. Click "Swap" and confirm in your wallet

## Providing Liquidity

1. Navigate to the "Liquidity" tab
2. Select a token pair (must include FeelsSOL)
3. Set your price range
4. Enter the amounts to deposit
5. Click "Add Liquidity" and confirm

## Understanding Fees

Feels Protocol uses a dynamic fee model based on:
- Market volatility
- Liquidity depth
- Time-weighted factors

Typical fees range from 0.01% to 0.3% depending on conditions.
